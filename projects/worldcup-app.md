# WorldCup Prediction App

## What it is

An internal web app built to turn a simple World Cup “office pool” into a full digital experience.

Instead of using a shared Excel, the goal was to create something engaging, social and easy to use for a team environment.

The entire product was built in less than a week, mainly using AI tools.

---

## Problem

A team initiative to run a World Cup pool was originally planned using a shared Excel.

This approach had multiple issues:

- Manual management of predictions and scoring  
- High risk of errors  
- No real-time visibility  
- Low engagement and interaction between participants  

Overall, the experience would have been static and difficult to manage.

---

## Solution

I built a complete web application to replace the Excel-based approach.

The app enables:

- Match predictions for all participants  
- Real-time leaderboard and ranking  
- Visibility of other users’ predictions  
- Scoring system applied automatically  
- Chat between participants to increase interaction  
- Admin panel to manage results and system updates  

The focus was not only functionality, but creating a dynamic and social experience.

---

## Why it matters

This project shows how a simple internal need can be turned into a full product experience.

Instead of manually managing a process, the solution creates:

→ engagement  
→ competition  
→ team interaction  

It transforms a coordination task into a shared experience.
