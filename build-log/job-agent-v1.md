# JobAgent — from automation to system thinking

## What I did

Built an AI job search workflow by adapting an existing tool (ApplyPilot).

Started using it in a real context and refining it based on daily usage.

Defined a broader direction for the project: moving beyond applications into a more complete job search system.

## What I learned

Job search is not only about applying.

Networking is a key component, and most tools ignore it.

Building a useful system often starts by adapting existing tools and then identifying what is missing.

- Explore how to integrate recruiter search and outreach
